# ИИ Юрист «Галина» — прототип

Кликабельный прототип мобильного приложения. Функционала нет — это обёртка для показа
интерфейса: все ответы заглушечные («Тут будет отвечать ваше апи»).

## Как посмотреть

Статика без сборки — деплоится на Vercel как есть, точка входа `index.html`.
Локально можно просто открыть `index.html` в браузере.

Приложение открывается на **онбординге**, кнопка «Начать» ведёт на главный экран.
На телефоне занимает весь экран, на десктопе показывается по центру в рамке телефона.

Рядом лежит `prototype.html` — та же версия одним файлом (стили, скрипты и картинки
вшиты внутрь), удобно переслать без сервера.

## Экраны

| Экран | Что там |
|---|---|
| Онбординг | стартовый: иллюстрация Галины, приветствие, кнопка «Начать» |
| Основной | баланс, приветствие, три режима, поле ввода, «Созвон» |
| Чат | сообщения, «Думаю…», анимация создания документа |
| Созвон | 3 состояния: слушаю / микрофон выключен / Галина отвечает |
| Сайдбар | чаты, договоры, Консультация, Новости, Отзывы, Поддержка |
| Разделы меню | наполнение под каждый пункт сайдбара |
| Шторка тарифа | «1 запрос = 3 рубля» — по тапу на баланс |

## Что кликается

- **Начать** — переход с онбординга на основной экран
- **Найти закон / Составить договор / Проверить договор** — выбор режима,
  в поле ввода подставляется зелёный префикс
- **Enter** — отправка. Ответ: «Думаю…» → текст-заглушка.
  В режиме «Составить договор» между ними — анимация создания документа
- **Созвон** — экран звонка, состояния меняются сами по кругу,
  тап по кнопке выключает микрофон
- **30₽** — шторка с тарифом
- **Иконка панели** — сайдбар, оттуда открываются все разделы
- **Esc** — закрыть любое окно

## Сборка однофайловой версии

`prototype.html` собирается из разбитой версии: CSS и JS вставляются инлайном,
картинки — в base64. После правок в `index.html` / `styles.css` / `app.js`
пересоберите его, иначе изменения в нём не появятся.
